# ─── Native components pip CANNOT install (install separately if needed) ───
# Packet capture (the `cap` + `dns` features) needs a libpcap/Npcap driver,
# which is a kernel component — pip only installs Python packages, so this
# MUST be installed by hand:
#   Windows : install Npcap  ->  https://npcap.com   (enable "WinPcap API-compatible")
#   macOS   : already built in (libpcap + tcpdump ship with the OS)
#   Linux   : already built in (apt install libpcap0.8 if missing)
#
# SNMP per-port counters (optional) use the net-snmp command-line tools:
#   Windows : install Net-SNMP and add it to PATH  ->  https://www.net-snmp.org
#   macOS   : brew install net-snmp
#   Linux   : apt install snmp
